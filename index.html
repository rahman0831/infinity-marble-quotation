🚀 Deploy to GitHub Pages — Step by Step

Make this app live on the web for free
1
Download this app file

Click ⬇ Export in the top header — this downloads the index.html file to your computer.

Save it somewhere easy to find, e.g. your Desktop.

2
Create a GitHub account

Go to github.com and sign up for a free account.

Choose a username like infinitymarble — this will appear in your web address.

3
Create a new repository

After logging in, click the + button (top right) → New repository.

Name it: infinity-marble-quotation
Set it to Public ✓
Check "Add a README file" ✓
Click Create repository.

4
Upload the index.html file

Inside your new repository, click Add file → Upload files.

Drag your index.html into the upload box.
Scroll down and click Commit changes.

5
Enable GitHub Pages

In your repository, click Settings (top tabs).
Scroll down to Pages in the left sidebar.
Under Source, select Deploy from a branch.
Branch: main · Folder: / (root)
Click Save.

✓
Your app is live!

Wait 2–3 minutes, then visit:
https://USERNAME.github.io/infinity-marble-quotation Replace USERNAME with your GitHub username.

Share this link with your team to use the app from any device!

⚠️
Important: API Key on GitHub Pages

When using the app on GitHub Pages, each user must enter their own Anthropic API key in the AI panel — the key is saved only in their browser. The key is never stored on any server. For a team setup where you want one shared key, you will need a simple backend proxy — ask Claude for help setting up a Cloudflare Worker or Vercel function to handle this securely.
